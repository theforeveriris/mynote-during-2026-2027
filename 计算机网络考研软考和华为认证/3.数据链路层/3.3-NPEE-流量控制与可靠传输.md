## 1.数据链路层的流量控制

较高的发送速度和较低的接收能力的不匹配，会造成传输出错。

| 协议             | 发送窗口 | 接收窗口 | 特点                                 |
| -------------- | ---- | ---- | ---------------------------------- |
| **停止-等待协议**    | 1    | 1    | 每发送完一个帧就停止发送，等待对方的确认，在收到确认后再发送下一个帧 |
| **滑动窗口协议**     | >1   | 1    | 连续发送多个帧                            |
| ├─ 后退N帧协议（GBN） | >1   | 1    | 出错后重传从出错帧开始的所有帧                    |
| └─ 选择重传协议（SR）  | >1   | >1   | 只重传出错帧                             |
