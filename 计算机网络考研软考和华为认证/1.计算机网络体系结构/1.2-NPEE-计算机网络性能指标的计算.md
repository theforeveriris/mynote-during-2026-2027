| 指标          | 符号 | 定义               | 单位                    |
| ----------- | -- | ---------------- | --------------------- |
| **速率/数据率**  | R  | 单位时间内传输的数据量      | b/s, kb/s, Mb/s, Gb/s |
| **带宽**      | B  | 单位时间内信道能通过的最高数据率 | b/s（或Hz）              |
| **吞吐量**     |    | 单位时间内通过某网络的实际数据量 | b/s                   |
| **时延**      |    | 数据从网络一端到另一端所需时间  | s, ms, μs             |
| **时延带宽积**   |    | 传播时延 × 带宽        | bit                   |
| **往返时延RTT** |    | 发送方发送数据到收到确认的时间  | s, ms                 |
| **利用率**     | U  | 信道利用率/网络利用率      | 百分比                   |

## 1.时延的计算

总时延 = 发送时延 + 传播时延 + 处理时延 + 排队时延

$$d_{\text{total}} = d_{\text{trans}} + d_{\text{prop}} + d_{\text{proc}} + d_{\text{queue}}$$

$$d_{\text{trans}} = \frac{L}{R}$$

$$d_{\text{prop}} = \frac{D}{V}$$

其中 L 为数据帧长度（bit），R 为发送速率（b/s），D 为信道长度（m），V 为传播速率（b/s，或m/s）。

| 时延类型     | 公式                       | 影响因素                              |
| -------- | ------------------------ | --------------------------------- |
| **发送时延** | = 数据帧长度(bit) / 发送速率(b/s) | 发送速率、数据长度                         |
| **传播时延** | = 信道长度(m) / 传播速率(m/s)    | 信道长度、介质（光速约3×10⁸m/s，电缆中约2×10⁸m/s） |
| **处理时延** | 无固定公式                    | 设备性能                              |
| **排队时延** | 无固定公式                    | 网络拥塞程度                            |

## 2.速率的计算


$R = \frac{\text{传输的比特数}}{\text{时间}}$

## 3.带宽的计算


$$B = \text{信道能通过的最高数据率} \quad (\text{单位：b/s})$$

$$B = \frac{Q}{T}$$

> Q = 数据量（Data Amount，单位：bit）

>T  = 时间（Time，单位：s）




## 4.时延带宽积的计算


$$d_{\text{prop}} \times B = \text{传播时延} \times \text{带宽} \quad (\text{单位：bit})$$

$$\text{Delay-Bandwidth Product} = d_{\text{prop}} \times R = \frac{D}{V} \times R$$

## 5. RTT（往返时延）的计算

$$\text{RTT} \approx 2 \times d_{\text{prop}}$$


## 6.奈奎斯特定律（无噪声信道）

$$R_{\max} = 2B \cdot \log_2{V}$$
> Rmax = 最大数据率（b/s）
> B  = 信道带宽（Hz）
> V  = 信号离散等级数（电平数） 
> 当V=2 （二进制信号）时，Rmax = 2B 


## 7.香农定律（有噪声信道）

$$R_{\max} = B \cdot \log_2{(1 + S/N)}$$
> Rmax = 最大数据率（b/s）
> B  = 信道带宽（Hz）
> S  = 信号功率（W）
> N  = 噪声功率（W）
