物理层设备的核心功能：信号的再生、放大、转发。它们工作在OSI最底层，只能处理比特流，无法识别数据链路层的帧结构。

## 1.中继器（Repeater）

中继器是最简单的物理层设备，功能单一：将接收到的衰减信号进行再生、放大后再发送出去。当数字信号在物理媒介中传输一定距离后，会因电阻、噪声、干扰等因素而逐渐衰减、失真。中继器的作用就是在信号尚未完全丢失前，对其进行再生、放大，以保持信号的完整性和质量。

 物理层"电气特性"规定——0.5~1.5V 是低电平，4.5~5.5V 是高电平，不符合此标准的信号视为无效，中继器接收到信号后，会将低电平整形为1V，将高电平整形为5V，然后再输出

| 特性       | 说明                        |
| -------- | ------------------------- |
| **工作层次** | 仅物理层                      |
| **通信方式** | 半双工（Half Duplex）             |
| **处理对象** | 比特流（无法识别帧、MAC地址）          |
| **信号处理** | 再生（Reshape）+ 放大（Amplify）  |
| **延迟**   | 极小（物理层处理）                 |
| **端口数**  | 通常2个（一进一出）                |
| **连接网段** | 两端必须是**相同类型**、**相同速率**的网段 |

> 传统以太网（10BASE5/10BASE2）使用中继器时的限制：
> 5  ── 最多5个网段（Segment）
> 4  ── 最多4个中继器
> 3  ── 其中只有3个网段可以挂接设备（其余2个用于延长）
> 目的：防止信号经过过多中继后累积延迟过大，导致冲突检测失效。

局限性：
| 局限           | 说明                   |
| ------------ | -------------------- |
| **不能连接不同速率** | 两端必须是相同速率（如都是10Mbps） |
| **不能连接不同协议** | 两端必须是相同协议（如都是以太网）    |
| **不能分割冲突域**  | 所有网段属于同一个冲突域         |
| **不能过滤流量**   | 所有信号无脑转发，包括错误帧       |
| **有距离限制**    | 受5-4-3规则约束           |

## 2.集线器（Hub）

集线器（Hub） 也称为多端口中继器（Multiport Repeater），它扩展了中继器的功能，允许多个设备通过一个中心点互联，形成星型拓扑结构。

| 特性       | 说明                    |
| -------- | --------------------- |
| **工作层次** | 仅物理层                  |
| **端口数**  | 4/8/16/24/48口等        |
| **工作方式** | **半双工**（同一时间只能收或发）    |
| **信号转发** | **广播式**：向所有端口转发（除源端口） |
| **带宽共享** | 所有端口共享总带宽             |
| **冲突域**  | **所有端口属于同一个冲突域**      |

带宽计算：

| 场景                | 计算方式                   |
| ----------------- | ---------------------- |
| **N端口集线器，总带宽B**   | 每个端口平均带宽 = B/N         |
| **10Mbps Hub，8口** | 理论每口1.25Mbps，实际更低（有冲突） |

局限性：
| 局限          | 详细说明                        |
| ----------- | --------------------------- |
| **不能分割冲突域** | 所有端口在同一个冲突域，CSMA/CD机制下冲突概率大 |
| **广播风暴**    | 所有信号广播，网络流量大时性能急剧下降         |
| **半双工限制**   | 不能同时收发，带宽利用率低               |
| **无过滤能力**   | 无法识别MAC地址，不能定向转发            |
| **无VLAN支持** | 无法隔离广播域                     |
| **安全性差**    | 所有端口收到所有数据，易被嗅探             |

与中继器对比：

| 对比项      | 中继器       | 集线器              |
| -------- | --------- | ---------------- |
| **端口数**  | 2个        | 多个（4/8/16/24/48） |
| **本质**   | 单端口放大器    | 多端口中继器           |
| **功能**   | 信号再生、延长距离 | 信号再生+多端连接        |
| **连接方式** | 点对点       | 星型拓扑中心           |
| **应用场景** | 两网段间延长    | 多设备星型组网          |
| **冲突域**  | 不分割       | 不分割（所有端口同一冲突域）   |
| **广播域**  | 不分割       | 不分割              |
